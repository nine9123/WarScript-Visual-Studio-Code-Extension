# WarScript — VS Code Extension

Syntax highlighting, bracket matching, code folding, and auto-indentation for WarScript `.ws` files.

## Features

- Syntax highlighting for all WarScript keywords, operators, strings, numbers, and comments
- Function and class definition highlighting
- Function call highlighting (both standalone and class method calls via `::`)
- Bracket matching and auto-closing for `[]`, `{}`, `()`, and `""`
- Code folding between `if/fun/class/loop/begin` and `end`
- Auto-indentation on block keywords
- Comment toggling with `Cmd+/` / `Ctrl+/`
